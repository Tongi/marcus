# SupremeRestocks
Script that monitors supremenewyork for items that have restocked.

## Installation

SupremeRestocks requires [Node.js](http://nodejs.org/).

Setup:

```sh
$ git clone https://github.com/Asteezy24/SupremeRestocks.git
$ cd SupremeRestocks
$ npm install
```


Run After Setup:

```sh
$ node supreme.js
```
